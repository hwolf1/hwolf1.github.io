hexo+mellow+github创建个人博客

一、安装hexo

​ 新建hexo文件夹，安装Node.js和Git，安装成功后，右键该文件夹Git Bash Here，打开命令窗口，下载hexo安装包，npm install hexo-cli -g ，初始化hexo，hexo init，安装deployer插件npm install hexo-deployer-git --save，配置hexo根目录下的_config.yml：repo: git@github.com:hwolf1/hwolf1.github.io.git ，设置好github仓库地址，接下来设置SSH KEY:,$ ssh-keygen -t rsa -C "你的邮箱",将生成的rsa.pub文件内容复制到github上的setting-rsa。

二、安装mellow主题

​ 安装需确认你的 Hexo 版本在 3.0 以上，以及 Node 版本为 6.x 以上，在 Hexo 根目录，执行以下命令。

git clone git@github.com:codefine/hexo-theme-mellow.git themes/mellow，

将hexo/_config.yml中的更改如下字段：

theme: mellow

安装mellow必备的依赖项：

  • hexo-generator-search - local search搜索

  • hexo-generator-topindex - 文章置顶

  • hexo-helper-qrcode - 二维码分享

  • hexo-renderer-less - less解析器

    npm install --save hexo-generator-search@2.1.1 hexo-generator-topindex@0.3.0 hexo-helper-qrcode@1.0.2 hexo-renderer-less@0.2.0

设置好后，根据hexo/theme/mellow/下的_config.yml中文注释自行配置主题

三、文本编辑

​ 文本编辑采用Typora软件进行编辑，对于图片编辑，参考博客，首先把主页配置文件_config.yml 里的这个选项设置为post_asset_folder:true，其次hexo目录下执行这样一句话npm install hexo-asset-image --save，这是下载安装上传本地图片的插件，使用：hexo n 文件，生成文件.md并且在source/ _posts生成对应的文件夹，将图片放到对应的文件夹，在md文件中插入。

四、发布博客

​ 在TYpora编辑好文档后，在bash中执行：hexo clean ---> hexo d -g,编译并且上传代码，将图片放到对应的文件夹，直接调用。示例博客内容如下：
